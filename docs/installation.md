# Installation

## Prerequisites
- Python 3.10+
- Git

## Steps
1. Clone the repository:
   \`\`\`bash
   git clone https://github.com/your-org/storysync.git
   cd storysync
   \`
2. Install dependencies:
   \`\`\`bash
   pip install -r requirements.txt
   \`
3. Configure \`examples/config.yaml\` with your API keys.
